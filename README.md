# markdown-alert-example
A small repository to illustrate a small problem with Markdown Preview for Github Alerts

First, we have a warning that renders as expected:

> [!WARNING]
> If you put a blank line after the `[!WARNING]` tag, GitHub will render things
> just fine.

Next, we have a warning that does not render as expected:

> [!WARNING]
>
> If you do not put a blank line after the `[!WARNING]` tag, Markdown Preview for Github
> Alerts will not render things correctly.
